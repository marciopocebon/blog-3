Fim. Un vilsualizador de imágenes liviano para terminal

Instalación

sudo apt-get install fim

Abrir Imagen

Con zoom automático

fim imagen.jpg

fim -a imagen.jpg

Carpeta

Ver todas las imágenes de una carpeta

fim carpeta/

Carpeta recursiva

Para abrir las imagenes de la carpeta y subcarpetaras recursivamente y despues ordenarlas:

fim -R carpeta/ --sort

En la Terminal

Para ver una imágen renderizada en formato ASCII, desde la propia terminal:

fim -t imagen.png

Atajos

PageUp/Down : Prev/Next image
+/- : Zoom in/out
a : Autoscale
w : Fit to width
h : Fit to height
j/k : Pan down/up
f/m : flip/mirror
r/R : Rotate (Clock wise and ant-clock wise)
ESC/q : Quit
