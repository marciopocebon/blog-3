Acceder dentro de un Docker como usuario root

Tiempo estimado de lectura: 1 minutos. 135 palabras. docker

Vamos a acceder a un contenedor como usuario root. Para ello tenemos que utilizar la opción -u seguido del usuario, en este caso root. Por ejemplo, para acceder dentro de un contenedor llamado nextcloud, utilizaríamos:

docker exec -it -u root nextcloud /bin/bash
